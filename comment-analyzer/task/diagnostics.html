Диагностика и нейтрализация ошибок

Для диагностики и нейтрализации ошибок был использован метод Айронса.
Его суть заключается в отбрасывании символов строки, приводящей к ошибке, и продолжении сканирования строки без учета этих символов.
При обнаружении символа, которого нет в алфавите, в программе будет выведено предупреждение о его наличии, включая его позицию.

Согласно заданию, требуется реализовать нейтрализацию ошибок синтаксиса с помощью метода Айронса.
Алгоритм Айронса для нейтрализации синтаксических ошибок выглядит следующим образом:
При обнаружении ошибки (во входной строке в процессе разбора встречается символ, который не соответствует ни одному из ожидаемых символов),
входная последовательность символов выглядит следующим образом: Tt, где T – следующий символ во входном потоке (ошибочный символ),
t – оставшаяся во входном потоке последовательность символов после T. Алгоритм нейтрализации состоит из следующих шагов:

  1. Определяются недостроенные кусты дерева разбора
  2. Формируется множество остаточных символов недостроенных кустов дерева разбора. Обозначим его L.
  3. Из входной последовательности символов удаляется следующий символ до тех пор, пока следующий символ в последовательности не сможет быть выведен из какого-нибудь остаточного символа недостроенных кустов
  4. Находим недостроенный куст, который привел к ошибочной ситуации
В итоге мы сможем понять к какому кусту следует привязывать оставшуюся последовательность символов после удаления ошибочной части символов. Для автоматных грамматик метод Айронса имеет одну существенную особенность: при разборе последовательности символов в дереве разбора всегда будет не достроен только один куст. Поэтому привязка оставшейся части символов после удаления ошибочной последовательности производится однозначно. В итоге метод Айронса для автоматных грамматик можно свести к последовательному удалению ошибочных символов, пока не встретится символ из алфавита.
